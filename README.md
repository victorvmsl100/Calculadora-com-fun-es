# 🧮 Calculadora com Funções em Python

Este projeto foi desenvolvido durante meus estudos de Python com o objetivo de praticar a criação de funções, estruturas condicionais e organização do código.

## 🚀 Funcionalidades

- ➕ Soma
- ➖ Subtração
- ✖️ Multiplicação
- ➗ Divisão
- ✅ Tratamento para divisão por zero
- 📋 Menu interativo para escolha da operação

## 🛠️ Tecnologias utilizadas

- Python 3

## 📚 Conceitos praticados

- Funções (`def`)
- Parâmetros
- `return`
- Estruturas condicionais (`if`, `elif` e `else`)
- Entrada e saída de dados (`input` e `print`)
- Organização e reutilização de código

## ▶️ Como executar

1. Clone este repositório:
   ```bash
   git clone https://github.com/victorvmsl100/Calculadora-com-Funcoes.git
   ```

2. Entre na pasta do projeto:
   ```bash
   cd Calculadora-com-Funcoes
   ```

3. Execute o programa:
   ```bash
   python calculadora.py
   ```
